# Tech Stack: Chakra Energy

## Frontend
- **Languages:** HTML5, CSS3, JavaScript (ES6+).
- **Styling:** Custom Vanilla CSS, Font Awesome 4.7.0.
- **Fonts:** SugarMagic (Custom OTF/WOFF2).

## PWA & Web APIs
- **Service Workers:** `sw.js` for offline caching and resource management.
- **Manifest:** `manifest.json` for standalone mobile/desktop installation.
- **Web Speech API:** `window.speechSynthesis` for guided meditation narration.
- **Web Audio API:** For high-fidelity playback of chakra frequencies and bell sounds.

## Storage & Integration
- **Client-Side Storage:** `localStorage` for user preferences and session data.
- **Hardware Integration:** NFC API (potential support as indicated by `nfc.html`).

## Development Tools
- **Version Control:** Git.
- **Methodology:** Conductor (Track-based development).
